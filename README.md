## рЯФН What is ReviewGuard?

**ReviewGuard** is a full-stack Django web application that uses a custom **NLP rule-based engine** to detect whether a product review is genuinely written by a human вАФ or fabricated by a machine.

Every review gets scanned, scored, and labelled as:

| Label | Meaning |
|-------|---------|
| рЯ§Ц **CG** | Computer Generated вАФ Likely Fake |
| рЯІС **OR** | Original Review вАФ Human Written |

No ML models. No external APIs. Pure linguistic intelligence.

## рЯІ† How Detection Works

The core logic lives in `reviews/detector.py`. Each review is scored **0вАУ100** using 7 handcrafted linguistic rules:

```
Score вЙ• 45  вЖТ  рЯ§Ц CG (Fake)
Score < 45  вЖТ  рЯІС OR (Human)
```

| # | Rule | Signal |
|---|------|--------|
| 1 | Review has fewer than 8 words | рЯФі Fake |
| 2 | Contains generic phrases like *"highly recommend"*, *"great product"* | рЯФі Fake |
| 3 | Missing personal pronouns (*I*, *my*, *we*) | рЯФі Fake |
| 4 | Excessive `!!!` or ALL CAPS usage | рЯФі Fake |
| 5 | Suspiciously uniform sentence lengths | рЯФі Fake |
| 6 | Informal language (*lol*, *tbh*, *kinda*) | рЯЯҐ Human |
| 7 | Specific personal details (family, time, comparisons) | рЯЯҐ Human |
