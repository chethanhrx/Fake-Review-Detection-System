ReviewGuard

Intelligent Fake Review Detection System


Presented By: Chethan

Advanced Machine Learning & Web Security

The Problem

  • E-commerce platforms are flooded with Computer-Generated (CG) and fabricated reviews.
  • Fake reviews mislead consumers, skew product ratings, and damage brand trust.
  • Standard keyword blocking is ineffective against modern AI tools (like ChatGPT) writing fake text.
  • We need a system that understands human emotion, quirks, and context.

What is ReviewGuard?

A full-stack, secure platform that authenticates the validity of product reviews using a Hybrid NLP + ML Engine.


🤖 Detects Bots

Identifies repetitive phrasing, absence of personal pronouns, and generic templates.

🧑 Authenticates Humans

Recognizes conversational tone, specific sizing/usage context, and emotional nuance.

Platform Features

  • Dual Authentication System: Distinct, secure portals for standard Users and Administrators.
  • Admin Analytics Dashboard: Live tracking of user activity, total reviews, and systemic fake-vs-real ratios.
  • Real-time Detection: Analyzes text locally the moment a user submits the review.
  • Graceful Offline Mode: Can operate entirely offline via strict linguistic rule-engines if external ML frameworks fail.

🔴 Red Flags (Bot Patterns):

  • Hyper-uniform sentence lengths and structure
  • Over-indexing on marketing buzzwords (\"game-changer\", \"highly recommend\")
  • Zero first-person pronouns (I, my, mine)

🟢 Green Flags (Human Patterns):

  • Shorthand typing quirks (thx, gr8, emojis)
  • Detailed physical usage (\"wore them in the rain\", \"my daughter uses it for gym\")
  • Balanced critiques (not 100% purely positive)

Layer 2: DistilBERT Integration

Rule-based systems struggle with \"Grey Zone\" text.
We integrated HuggingFace Transformers to fix this.

Hybrid Architecture

ReviewGuard runs DistilBERT (a smaller, faster, cheaper NLP model) to classify subtext.

The Tensor probabilities are injected directly into the NLP Rule weights, creating an incredibly resilient, high-accuracy detector.
